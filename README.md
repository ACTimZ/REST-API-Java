REST API на Java. 28.04.2025. Тема: "Ювелирный магазин"

Сущности:

- Рабочий;
- Должности рабочих;
- Клиент;
- Контактная информация о рабочих и клиентах;
- Ювелирное украшение;
- Чек об оплате;
- Материал ювелирного изделия.

Для запуска проекта в терминале пишем:

# ./gradlew bootRun

или, если Windows:

# gradlew.bat bootRun
